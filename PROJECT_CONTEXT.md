# 편의점 보충진열 체크리스트

## 프로젝트 개요

편의점 보충진열 업무를 돕는 모바일 웹앱.

부족한 상품을 기록한 뒤 창고에서 한 번에 물건을 수집하여  
매대↔창고 왕복 횟수를 줄이는 것이 목적.

---

- Next.js
- React
- TypeScript
- LocalStorage
- Vercel

---

v1.2

## 실제 사용 환경

GS 편의점 야간 근무

사용 흐름

1. 매대를 순회
2. 부족 상품 등록
3. 창고 이동
4. 상품 수집
5. 매대 보충

## 실사용 검증 결과

### 효과 있었음

- 창고↔매대 왕복 감소
- 필요한 상품을 한 번에 수집 가능

### 문제점

- 체크리스트 작성 시간이 생각보다 길다
- 입력 과정이 병목
- 상품 식별이 어려운 경우 존재

---

## 현재 병목

상품 등록 과정

현재

상품명 입력  
→ 카테고리 선택  
→ 수량 입력  
→ 추가

목표

상품 등록 시간을 최대한 줄이는 것
