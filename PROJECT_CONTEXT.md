# 편의점 보충진열 체크리스트

## 프로젝트 개요

편의점 보충진열 업무를 돕는 모바일 웹앱.

매대를 순회하며 부족한 상품을 기록한 뒤 창고에서 필요한 상품을 한 번에 수집하여 매대↔창고 왕복 횟수를 줄이는 것이 목적이다.

현재 버전

실사용 중

### 최신 실사용 결과

- 사용자 상품 관리 기능 추가 완료
- 자동완성 데이터는 실제 사용량에 비례하여 지속적으로 향상됨
- 사용자 상품 수정/삭제 가능
- 체크리스트 저장 안정성 개선
- 자동완성 데이터 품질 관리 가능

---

### 현재 제품 방향

핵심 목표

- 체크리스트 작성 시간 단축
- 반복 입력 감소
- 실제 업무 흐름 유지
- 단순한 UI
- 유지보수하기 쉬운 구조

실사용 결과를 기준으로 기능 유지 여부를 결정한다.

실사용 빈도가 낮은 기능은 과감히 제거하거나 후순위로 이동한다.

반복되는 작업 데이터를 재활용하여 입력 시간을 줄이는 방향을 우선적으로 검토한다.
