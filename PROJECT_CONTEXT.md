# 편의점 보충진열 체크리스트

## 프로젝트 개요

편의점 보충진열 업무를 돕는 모바일 웹앱.

매대를 순회하며 부족한 상품을 기록한 뒤,

창고에서 필요한 상품을 한 번에 수집하여

매대↔창고 왕복 횟수를 줄이는 것이 목적.

### 실사용 환경

사용자는 편의점 2개 지점을 번갈아 근무한다.

현재는 점포 구분 기능 없이 사용 중이며,

점포별 재고 차이는 실제 사용 데이터를 확보한 후 검토한다.

## 핵심 기능

- 상품 추가
- 자동완성
- 카테고리 분류
- 수량 입력
- 창고 없음 체크
- 완료 체크
- LocalStorage 저장
- 미해결 상품 관리
- 지난 근무 상품 재확인
- 창고 없음 섹션 분리
- **사용자 상품 DB 자동 구축**
- **신규 상품 자동완성 반영**

---

## 사용자 상품 DB

자동완성 목록에 없는 상품을 입력하면

사용자 상품 DB(LocalStorage)에 자동 저장된다.

다음부터는 해당 상품이 자동완성 목록에 포함된다.

기본 상품 DB와 사용자 상품 DB를 함께 사용하여

실제 근무할수록 자동완성 품질이 향상된
