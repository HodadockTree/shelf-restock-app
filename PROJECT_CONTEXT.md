# 편의점 보충진열 체크리스트

## 프로젝트 개요

편의점 보충진열 업무를 돕는 모바일 웹앱.

매대를 순회하며 부족한 상품을 기록한 뒤 창고에서 필요한 상품을 한 번에 수집하여 매대↔창고 왕복 횟수를 줄이는 것이 목적이다.

## 장기 계획

기능은 실제 사용 후에만 추가한다.

후보 기능

- 최근 사용 상품
- 즐겨찾기
- 동일 상품 자동 합산
- 점포별 데이터 관리

Firebase 이전 가능성을 고려하여 데이터 접근은 API(`products.ts`)를 통해 수행한다.
