# 편의점 업무 도우미

## 프로젝트 개요

편의점 근무 중 반복적으로 발생하는 업무를 돕기 위한 모바일 웹앱.

초기 기능은 보충진열 체크리스트였으며, 매대를 순회하며 부족한 상품을 기록한 뒤 창고에서 필요한 상품을 한 번에 수집하여 매대↔창고 왕복 횟수를 줄이는 것을 목표로 했다.

v1.8부터는 보충진열 전용 앱에서 편의점 업무 도우미로 확장하며, 두 번째 기능으로 폐기 메모 MVP를 추가하였다.

이 프로젝트는 실제 편의점 근무 중 계속 사용하며, 필드 테스트 결과를 바탕으로 기능을 개선한다.

## 현재 제품 방향

현재 프로젝트는 보충진열 체크리스트에서 시작했지만, 장기적으로는 편의점 근무 중 반복적으로 발생하는 작은 업무들을 돕는 업무 도우미 앱을 목표로 한다.

v1.7까지의 핵심 방향은 보충진열 체크리스트의 입력 속도, 탐색성, 사용자 상품 관리 접근성 개선이었다.

v1.8부터는 기능 확장 단계로 진입하였다. 다만 한 번에 많은 기능을 넣기보다, 기존 개발 방식과 동일하게 MVP를 먼저 만들고 실제 근무 중 검증한 뒤 필요한 기능만 추가한다.

폐기 메모는 공책 수기 작성이나 사진만 따로 찍어두는 방식의 불편함을 줄이기 위한 기능이다. MVP에서는 사진 촬영, 간단한 메모, 저장, 목록 보기, 삭제만 제공한다.

아직 폐기 예정일, 위치, 수정, 완료 처리, 알림, 공유 기능은 추가하지 않았다. 실제 사용 후 필요성이 확인되면 다음 버전에서 검토한다.

## 기능 판단 기준

기능 유지 여부는 실제 편의점 근무 흐름을 기준으로 판단한다.

우선순위는 다음과 같다.

1. 실제 업무 시간 단축
2. 반복 입력 감소
3. 실제 업무 흐름 유지
4. 단순한 UI
5. 유지보수하기 쉬운 구조
6. LocalStorage 기반 구조에서 안정적으로 유지 가능할 것

기능을 추가할 때는 다음을 우선 검토한다.

· 실제 근무 중 조작 횟수를 줄이는가  
· 입력 또는 탐색 시간을 줄이는가  
· 기존 업무 흐름을 방해하지 않는가  
· UI가 복잡해지지 않는가  
· 저장 구조가 안정적인가  
· 실사용 전부터 과도하게 복잡한 기능은 아닌가

## 보충진열 기능 관련 판단

### 재확인 상품 기능

실제 업무 흐름과 맞지 않아 UI에서 숨김 처리하였다.

현재는 다음 버전에서 완전 제거 여부를 검토한다.

### 창고 없음 기능

창고 없음 여부를 즉시 확정하기 어려운 상황이 많고, 실제 업무 흐름에서 자주 사용되지 않아 UI에서 숨김 처리하였다.

현재는 다음 버전에서 완전 제거 여부를 검토한다.

### 직전 체크리스트

같은 점포에서 연속 근무할 때 반복 입력을 줄일 가능성이 있어 유지한다.

다만 실제 사용 빈도와 전체 불러오기 / 선택 추가 방식의 효용은 추가 검증이 필요하다.

### 체크리스트 검색

긴 체크리스트에서 원하는 상품을 찾기 위한 기능으로 추가하였다.

실제 근무 중 사용 빈도와 탐색 시간 감소 효과는 추가 필드 테스트가 필요하다.

## 폐기 메모 기능 관련 판단

### 현재 MVP 범위

· 사진 촬영 또는 선택  
· 간단한 메모 입력  
· 저장  
· 목록 보기  
· 삭제  
· 작성 시각 표시

### 현재 제외한 기능

· 폐기 예정일 별도 입력  
· 위치 선택  
· 상품명 별도 입력  
· 수정 기능  
· 완료 처리  
· 날짜별 묶기  
· 알림 기능  
· 다중 사진  
· 공유 기능  
· 클라우드 동기화

### 제외 이유

폐기 메모는 실제 사용 전이므로 처음부터 많은 입력 필드를 만들지 않는다.

현재 목표는 폐기 예정 상품을 빠르게 기록하고 다시 확인할 수 있는지 검증하는 것이다.

사진과 메모가 분리되어 관리되는 문제를 해결하는 것이 1차 목표이며, 세부 관리 기능은 실사용 후 필요성이 확인되면 추가한다.

## 다음 검증 항목

### 보충진열

체크리스트 검색

· 긴 체크리스트에서 원하는 상품을 더 빨리 찾을 수 있는지  
· 검색 중 체크 / 삭제 / 수량 변경이 자연스럽게 작동하는지  
· 검색 기능이 실제 근무 중 자주 사용되는지  
· 검색창이 화면을 복잡하게 만들지 않는지

자동완성 관리 흐름

· 자동완성에서 잘못 등록된 사용자 상품을 발견했을 때 관리 버튼을 사용하는지  
· `/user-products?keyword=상품명` 이동이 실제로 수정/삭제 시간을 줄이는지  
· 기본 상품 DB와 사용자 상품 DB 구분이 자연스러운지  
· 자동완성 목록에서 관리 버튼이 방해되지 않는지

PWA

· 홈화면 설치가 실제 접근성을 높이는지  
· 독립 실행 화면이 업무 도구처럼 느껴지는지  
· 모바일에서 앱 실행 흐름이 브라우저 접속보다 빠른지

직전 체크리스트

· 같은 점포 연속 근무에서 직전 체크리스트 재사용 빈도  
· 전체 불러오기와 선택 추가 중 어떤 방식이 더 편한지  
· 직전 체크리스트 개별 삭제가 필요한 상황이 얼마나 자주 발생하는지

### 폐기 메모

사진 촬영 흐름

· 모바일에서 카메라가 자연스럽게 열리는지  
· 사진 선택 과정이 번거롭지 않은지  
· 사진 미리보기가 충분히 잘 보이는지  
· 사진 저장 속도가 느리지 않은지

메모 입력

· 자유 메모 하나로 충분한지  
· 폐기 예정일을 별도 필드로 입력하고 싶은지  
· 위치를 별도 필드로 입력하고 싶은지  
· 상품명을 별도 필드로 입력하고 싶은지  
· 메모 입력이 수기 공책보다 편한지

목록 확인

· 저장된 메모를 다시 확인하기 쉬운지  
· 작성 시각 표시가 도움이 되는지  
· 오래된 메모가 쌓였을 때 불편한지  
· 삭제만으로 충분한지, 완료 처리가 필요한지

저장 안정성

· 새로고침 후에도 사진과 메모가 유지되는지  
· PWA로 실행해도 정상 작동하는지  
· 사진 여러 장 저장 시 속도나 용량 문제가 생기지 않는지
