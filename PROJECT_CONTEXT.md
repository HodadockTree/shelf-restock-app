# 편의점 보충진열 체크리스트

## 프로젝트 개요

편의점 보충진열 업무를 돕는 모바일 웹앱.

매대를 순회하며 부족한 상품을 기록한 뒤 창고에서 필요한 상품을 한 번에 수집하여 매대↔창고 왕복 횟수를 줄이는 것이 목적이다.

## 현재 제품 방향

v1.6의 핵심 방향은 신규 기능 확장이 아니라 반복 입력 감소이다.

실사용 결과, 자동완성 데이터는 누적될수록 효율이 높아졌지만 매 근무마다 비슷한 체크리스트를 다시 구성해야 하는 문제가 남아 있었다.

따라서 v1.6에서는 직전 체크리스트를 저장하고, 다음 근무에서 필요한 항목만 다시 불러오는 방식으로 입력 시간을 줄이는 데 집중하였다.

## 기능 판단 기준

기능 유지 여부는 실제 편의점 근무 흐름을 기준으로 판단한다.

우선순위는 다음과 같다.

1. 체크리스트 작성 시간 단축
2. 반복 입력 감소
3. 실제 업무 흐름 유지
4. 단순한 UI
5. 유지보수하기 쉬운 구조
