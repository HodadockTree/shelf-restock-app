# 편의점 보충진열 체크리스트

## 프로젝트 개요

편의점 보충진열 업무를 돕는 모바일 웹앱입니다.

매대를 순회하며 부족한 상품을 기록한 뒤, 창고에서 필요한 상품을 한 번에 수집하여 매대↔창고 왕복 횟수를 줄이는 것을 목표로 합니다.

이 프로젝트는 실제 편의점 근무에서 사용하며, 실사용 테스트 결과를 바탕으로 지속적으로 개선하고 있습니다.

## 기술 스택

- Next.js
- React
- TypeScript
- LocalStorage
- Vercel

향후 Firebase 이전 가능성을 고려하여 설계합니다.

## 핵심 기능

- 상품 등록
- 자동완성
- 카테고리 분류
- 필요 수량 입력
- 창고 없음 체크
- 체크리스트 완료
- LocalStorage 저장
- 미해결 상품 관리
- 지난 근무 상품 재확인
- 사용자 상품 DB 자동 구축
- 기본 상품 + 사용자 상품 통합 검색

## 기능 추가 원칙

새로운 기능은 반드시 실사용 테스트를 통해 필요성이 확인된 경우에만 추가합니다.

검토 기준

- 실제 업무 효율이 향상되는가?
- 입력 시간이 감소하는가?
- 반복 작업이 줄어드는가?
- 유지보수가 쉬운 구조인가?

개발 난이도보다 실사용 가치를 우선합니다.

코드 작성보다 설계와 구현 순서를 먼저 결정합니다.
