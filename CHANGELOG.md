### 결과

- 실사용 가능한 최소 기능 구현 완료

### ## v1.7 - 체크리스트 검색 및 상품 관리 UX 개선



### 목표

체크리스트가 길어졌을 때 원하는 상품을 빠르게 찾고, 자동완성에서 잘못 등록된 사용자 상품을 발견했을 때 관리 페이지로 자연스럽게 이동할 수 있게 한다.

또한 홈화면 설치와 독립 실행을 지원하여 실제 근무 중 앱 접근성을 높인다.

### 배경

v1.6 필드 테스트 결과, 수량 직접 입력과 UI 단순화는 긍정적인 개선으로 확인되었다.

반면 자동완성 목록에서 잘못 등록된 사용자 상품을 발견해도 바로 수정하거나 삭제할 수 없어 사용자 상품 관리 페이지로 이동한 뒤 다시 검색해야 하는 문제가 있었다.

또한 체크리스트가 길어질수록 원하는 상품을 찾기 어려운 문제가 계속 남아 있었다. 이미 완료 상품은 아래로 이동하고, 라면 / 과자 카테고리 구분도 적용되어 있었기 때문에 v1.7에서는 정렬 기능이 아니라 체크리스트 내 검색 기능을 추가하였다.

### 구현



#### 체크리스트 검색

- 체크리스트 내 검색 기능 추가
- 상품명 기준 필터링 적용
- 검색어 입력 시 현재 체크리스트에서 해당 상품만 표시
- 검색어 삭제 시 전체 체크리스트 복귀
- 검색 결과 없음 메시지 추가
- 검색 중 체크 / 삭제 / 수량 변경 가능
- 기존 완료 항목 아래 이동 유지
- 기존 라면 / 과자 카테고리 구분 유지



#### 사용자 상품 관리 접근성 개선

- `/user-products` 페이지 상단에 체크리스트 복귀 링크 추가
- 복귀 UI를 화살표 형태로 단순화
- 브라우저 뒤로가기에 의존하지 않고 체크리스트 화면으로 이동 가능



#### 자동완성 관리 흐름 개선

- 자동완성 후보에서 기본 상품 DB와 사용자 상품 DB 구분
- 사용자 상품 항목에만 관리 버튼 표시
- 기본 상품 DB 항목에는 관리 버튼 미표시
- 관리 버튼 클릭 시 `/user-products?keyword=상품명` 으로 이동
- 상품명에 공백이나 특수문자가 있어도 이동 가능하도록 처리
- 관리 버튼 클릭 시 체크리스트에 상품이 추가되지 않도록 분리



#### 사용자 상품 관리 페이지 keyword 검색

- `/user-products?keyword=상품명` 접근 시 검색창에 keyword 자동 입력
- 해당 keyword 기준으로 사용자 상품 목록 자동 필터링
- keyword가 없을 때는 기존 방식 유지
- 검색어 직접 수정 가능



#### 사용자 상품 카드 UX 개선

- 기존 하단 분리형 수정 폼 제거
- 상품 카드 내부에서 수정 모드가 열리도록 변경
- 수정 모드에서 상품명 input, 카테고리 select, 저장, 취소 버튼 표시
- 수정 중인 상품을 카드 안에서 바로 확인 가능
- 수정 / 삭제 / 저장 / 취소 버튼 스타일 개선
- 삭제 버튼은 위험 액션으로 구분하되 과하게 튀지 않도록 조정



#### 기본 PWA 적용

- manifest 추가
- 앱 이름 설정
- short name 설정
- start_url 설정
- scope 설정
- display: standalone 설정
- 앱 아이콘 추가
- 홈화면 설치 지원
- 독립 실행 지원
- 서비스워커와 오프라인 캐싱은 제외



### 결과

- 긴 체크리스트에서 특정 상품을 빠르게 찾을 수 있게 됨
- 검색 중에도 체크리스트 조작이 가능해 실제 업무 흐름을 유지함
- 자동완성에서 잘못 등록된 사용자 상품을 발견했을 때 관리 페이지로 바로 이동 가능
- 사용자 상품 관리 페이지에서 해당 상품이 검색된 상태로 열려 수정/삭제 흐름이 짧아짐
- 사용자 상품 수정 UI가 카드 내부로 통합되어 직관성이 향상됨
- 사용자 상품 관리 페이지의 복귀 흐름이 명확해짐
- 웹앱을 홈화면에 설치하고 앱처럼 실행할 수 있게 됨
- 오프라인 캐싱은 적용하지 않아 LocalStorage 기반 앱의 안정성을 유지함



### 검증

- 체크리스트 검색 정상 작동 확인
- 검색 중 체크 / 삭제 / 수량 변경 정상 작동 확인
- 검색 결과 없음 메시지 확인
- 기존 카테고리 구분 유지 확인
- 사용자 상품 관리 페이지 복귀 버튼 정상 작동 확인
