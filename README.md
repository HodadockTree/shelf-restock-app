# 편의점 보충진열 체크리스트

편의점 보충진열 업무를 빠르게 기록하기 위한 모바일 웹앱입니다.

실제 편의점 근무 중 사용하며 부족한 상품을 기록한 뒤, 창고에서 필요한 상품을 한 번에 수집하여 **매대↔창고 왕복 횟수를 줄이는 것**을 목표로 합니다.

---

## 주요 기능

- 상품 자동완성
- 사용자 상품 DB 자동 구축
- 체크리스트 관리
- 필요 수량 입력
- 창고 없음 체크
- 미해결 상품 관리
- 사용자 상품 관리 (검색 / 수정 / 삭제)
- LocalStorage 자동 저장

---
