# 편의점 보충진열 체크리스트

편의점 보충진열 업무를 빠르게 기록하기 위한 모바일 웹앱입니다.

실제 편의점 근무 중 사용하며 부족한 상품을 기록한 뒤, 창고에서 필요한 상품을 한 번에 수집하여 매대↔창고 왕복 횟수를 줄이는 것을 목표로 합니다.

## 주요 기능

- 상품 자동완성
- 사용자 상품 DB 자동 구축
- 체크리스트 관리
- 필요 수량 입력
- 체크리스트 내 검색
- 직전 체크리스트 저장 및 불러오기
- 사용자 상품 관리
  - 검색
  - 수정
  - 삭제
  - 자동완성에서 관리 페이지로 이동
- LocalStorage 자동 저장
- 기본 PWA 지원
  - 홈화면 설치
  - 앱 아이콘
  - 독립 실행

## 기술 스택

- Next.js
- React
- TypeScript
- LocalStorage
- PWA
- Vercel

## 프로젝트 원칙

기능 추가보다 실제 업무 효율을 우선합니다.

우선순위는 다음과 같습니다.

1. 체크리스트 작성 시간 단축
2. 반복 입력 감소
3. 실제 업무 흐름 유지
4. 단순한 UI
5. 유지보수하기 쉬운 구조

기능은 반드시 실제 편의점 근무에서 검증한 후 다음 버전을 설계합니다.
