# 편의점 보충진열 체크리스트

실제 편의점 야간 근무에서 사용하는 모바일 웹 체크리스트입니다.

매대를 순회하며 부족한 상품을 기록한 뒤, 창고에서 필요한 상품을 한 번에 수집하여 **매대 ↔ 창고 왕복 횟수를 줄이는 것**을 목표로 개발하고 있습니다.

이 프로젝트는 실제 근무 중 사용하며, 실사용 테스트 결과를 바탕으로 지속적으로 개선하고 있습니다.

---

## 주요 기능

- 상품 자동완성
- 카테고리 자동 입력
- 필요 수량 입력
- 창고 없음 체크
- 체크리스트 관리
- 미해결 상품 관리
- 사용자 상품 DB 자동 구축
- LocalStorage 자동 저장

---

## 사용 방법

1. 부족한 상품명을 입력합니다.
2. 자동완성에서 상품을 선택합니다.
3. 필요 수량을 입력합니다.
4. 체크리스트에 추가합니다.
5. 창고에서 상품을 수집합니다.
6. 보충이 완료되면 체크합니다.
