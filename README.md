# 편의점 보충진열 체크리스트

편의점 매대 보충진열 업무를 빠르게 기록하기 위한 모바일 웹 체크리스트입니다.

## 주요 기능

- 상품 자동완성
- 카테고리 자동 선택
- 보충진열 체크리스트 관리
- 진행률 표시
- 브라우저 자동 저장 (LocalStorage)

## 사용 방법

1. 상품명 입력
2. 자동완성 상품 선택
3. 필요 시 메모 입력
4. 체크리스트에 추가
5. 보충 완료 후 체크

## 기술 스택

- Next.js
- React
- TypeScript

## 개발 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속
